# WhatsApp AI Agent

## Features
- WhatsApp QR Login
- Gujarati, Hindi, English AI Replies
- Auto reconnect
- Railway compatible
- Replit compatible

## Setup

### 1. Upload to GitHub

### 2. Deploy on Railway or Replit

### 3. Add Environment Variable

GEMINI_API_KEY=YOUR_API_KEY

### 4. Start

npm install
npm start

### 5. Scan QR

Open WhatsApp:
Linked Devices → Link Device
